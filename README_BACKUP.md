# 项目备份说明

## 备份脚本

项目包含一个自动备份脚本 `backup_script.sh`，用于备份所有重要的文件和数据。

### 备份内容

- Python源代码文件
- 配置文件
- 数据文件（包括数据库）
- 日志文件
- 前端文件
- AutoX脚本
- 模板文件
- K线缓存
- 环境配置文件

### 运行备份

```bash
./backup_script.sh
```

### 备份存储位置

备份文件存储在 `备份文件/` 目录中，每个备份都有唯一的时间戳。

## 恢复备份

要恢复备份，只需将备份文件复制回项目根目录即可。

## 自动清理

备份脚本会自动清理30天前的备份文件，保持备份目录整洁。
